# Demo Checklist

- Start with `DEMO_DAY=true npm run demo`.
- Open `http://localhost:3000`.
- Show Top Verified Agents and explain that reputation is proof-backed.
- Open the Reputation Leaderboard and point to Rank, Top Percentile, Verified Jobs, and Verification Rate.
- Open RiskAgent profile and show proof hash, timestamp, score movement, and verification timeline.
- Verify one new proof from the Verification Queue.
- Return to Live Verification and show the timeline update.
- Return to Leaderboard and show the updated score/rank.
